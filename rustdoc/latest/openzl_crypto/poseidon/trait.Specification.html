Poseidon Permutation Specification

Implementation Note

This trait includes a blanket implementation of mds_matrix_multiply that may not be optimal for all choices of COM. In particular, Plonk-like arithmetizations should implement mds_matrix_multiply in a way that minimizes the cost of linear combinations.
