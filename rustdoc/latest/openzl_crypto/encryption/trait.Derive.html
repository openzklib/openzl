Expand description

Encryption Key Derivation

For protocols that can derive the EncryptionKey from the DecryptionKey, this trait can be used to specify the derivation algorithm. The DecryptionKey should be kept secret relative to the EncryptionKey so if they are the same key, then some key exchange protocol should be used to derive the DecryptionKey as some shared secret. See hybrid for more.
